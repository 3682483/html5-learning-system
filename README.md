# HTML5 闯关学习系统

一个交互式的 HTML5 编程学习平台，通过闯关模式帮助初学者掌握 HTML5 基础知识。

## 功能特点

- **闯关式学习**：10+ 个循序渐进的学习关卡
- **实时预览**：内置代码编辑器，支持实时预览效果
- **按行验证**：智能对比代码，精准定位错误
- **全屏模式**：支持全屏预览，方便学习
- **学习计时**：记录每个关卡的学习时间
- **进度保存**：自动保存学习进度

## 学习内容

| 周次 | 内容 |
|------|------|
| 第 1 周 | HTML 基础与版权信息 |
| 第 10 周 | BOM 与 DOM 动态操作 |

## 使用方法

1. 直接在浏览器中打开 HTML 文件
2. 阅读左侧关卡提示
3. 在右侧编辑器中编写代码
4. 点击「按行验证对比」检查答案
5. 通过验证后解锁下一关

## 技术栈

- HTML5
- CSS3

双击任意 HTML 文件即可在浏览器中运行，无需任何服务器。
